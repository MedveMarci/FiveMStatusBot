- .env fileban tudod konfigurálni a botot
- https://discord.com/developers/applications linken tudsz discord botot csinálni
- A StatusChannel-hez annak a csatorna ID-jét írd amelyik csatornába szretnéd hogy a státusz üzenetet küldje a bot. A BOT MINDEN INDULÁSNÁL KITÖRÖL ONNAN MINDENT
